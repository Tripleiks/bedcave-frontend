# Building a Budget Homelab Server

Creating a homelab doesn't have to break the bank. Here's how to build a capable server for under $500.

## Why a Homelab?

- **Hands-on Learning**: Practice real-world IT skills
- **Test Environment**: Safely experiment before production
- **Self-Hosting**: Run your own services (Nextcloud, Plex, etc.)
- **Privacy**: Keep your data under your control

## Recommended Budget Build

### Hardware Specs

| Component | Recommendation | Price |
|-----------|---------------|-------|
| CPU | Intel i5-10400 or AMD Ryzen 5 3600 | $150 |
| RAM | 32GB DDR4 (2x16GB) | $80 |
| Storage | 256GB NVMe + 2x4TB HDD | $120 |
| Motherboard | B460 or B550 (used) | $60 |
| PSU | 400W 80+ Bronze | $40 |
| Case | Any ATX case | $50 |

**Total: ~$500**

## Software Stack

1. **Proxmox VE** (Type 1 Hypervisor)
2. **Docker** for containerized services
3. **TrueNAS** or **OpenMediaVault** for storage

## First Services to Run

- **Pi-hole**: Network-wide ad blocking
- **Home Assistant**: Smart home automation
- **Vaultwarden**: Password manager
- **Jellyfin**: Media server

> Start small and expand as you learn. The best homelab is one that you actually use.

Happy building! 🏗️
